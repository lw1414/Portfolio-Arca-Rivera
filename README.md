# Portfolio — Aernest Lean Rivera

Personal portfolio of **Aernest Lean Rivera** — Licensed Electronics Engineer, Senior Hardware
Design Engineer, and builder of web & AI products. Currently training at **Arca PH**.

🔗 Featured project (live): **Tester.io** — https://arca-academy-activities-training-pv.vercel.app/

## Pages
- **`index.html`** — landing page
- **`about.html`** — full portfolio: hero, skills, experience, project highlights (Tester.io reel +
  grid), PCB & hardware design, education, and contact.

## Stack
Single-file static HTML/CSS/JS — no build step. Inter + JetBrains Mono, a teal-on-slate design
system, IntersectionObserver scroll reveals, and an FFmpeg-built project reel.

## Run locally
```bash
node serve-portfolio.mjs   # serves on http://localhost:3010
```
…or just open `about.html` in a browser.
